# visit-gent-dialogflow-fulfillment

## Installation

* clone the repository
```bash
git clone https://github.com/lab9k/visit-gent-dialogflow-fulfillment.git
```

* install dependencies
```bash
npm install
```

* create env file and edit the values as needed
```bash
mv ./env.local ./.env
```

* start the server
```bash
npm start
```
